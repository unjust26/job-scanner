# Job Scanner

Upload your resume and find matching job listings automatically.

## Tech Stack

- **React 19** + **TypeScript** — Frontend
- **Vite** — Build tooling
- **Convex** — Real-time backend & database
- **Tailwind CSS** — Styling
- **shadcn/ui** — UI components

## Getting Started

```bash
# Install dependencies
bun install

# Start the backend
bunx convex dev

# Start the frontend (in another terminal)
bun run dev
```
