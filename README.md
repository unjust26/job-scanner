# Job Scanner

Upload your resume and find matching job listings automatically.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** — Build tooling
- **Convex** — Real-time backend & database
- **Tailwind CSS** — Styling
- **shadcn/ui** — UI components

## Getting Started

```bash
bun install
bunx convex dev
bun run dev
```
